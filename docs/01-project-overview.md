# BioForge CCD Studio 프로젝트 개요

## 목표

바이오공정 제조라인 개념설계 패키지(CCD Package) 작성을 자동화하는 웹서비스의 초기 프로토타입을 구축한다.

이 프로토타입은 다음 질문에 답하기 위한 목적을 가진다.

- 입력 데이터를 어떤 방식으로 받을 것인가
- 엑셀 기반 입력과 화면 기반 직접 입력을 어떻게 공존시킬 것인가
- LLM 분석 결과를 어떤 형태의 CCD 패키지로 보여줄 것인가

## 이번 단계에서 구현한 범위

- 랜딩 페이지 1개
- 워크스페이스 페이지 1개
- 입력 방식 선택 UI
- 로컬 파일 업로드 UI
- Google Drive 불러오기용 모의 UI
- 직접 옵션 입력 폼
- 시트형 입력 미리보기 테이블
- CCD 패키지 초안 생성 데모
- 요약 TXT 다운로드
- 입력 시트 CSV 다운로드

## 이번 프로토타입의 핵심 가정

1. 실제 엑셀 입력 포맷은 아직 확정되지 않았다.
2. 따라서 업로드 기반과 폼 기반 입력을 동시에 검토할 수 있어야 한다.
3. 현재 단계에서는 실제 LLM API 호출 없이도, 사용자 흐름과 결과 구조를 먼저 검증하는 것이 중요하다.
4. 실제 Google Drive 연동은 차후 OAuth 및 Picker 연동으로 연결한다.

## 파일 구조

- `index.html`: 서비스 소개 및 진입 페이지
- `workspace.html`: 입력, 시트 미리보기, 결과 생성 페이지
- `styles.css`: 공통 스타일
- `script.js`: 업로드, 폼 동기화, 결과 생성, 다운로드 로직

## 기대 효과

- 사용자와 빠르게 화면을 보며 입력 방식 협의 가능
- CCD 패키지 산출물 구조를 화면에서 먼저 검증 가능
- 향후 Excel parser / LLM / Drive 연동 포인트를 명확히 분리 가능
