BIOPROCESS CONCEPT DESIGN AUTOMATION

엑셀과 입력 폼만으로
CCD 패키지를 자동 생성하는 제조라인 설계 워크스페이스

바이오공정 제조라인 개념설계 데이터를 업로드하거나 직접 입력하면, LLM이 핵심 조건을 정리하고 CCD 패키지 형태의 결과물을 제안하는 서비스 프로토타입입니다.

입력 워크스페이스로 이동 작동 방식 보기
입력 방식 Drag & Drop / Local File / Google Drive / Direct Form

엑셀 원본이 아직 고정되지 않아도, 동일 화면에서 실험적으로 입력 구조를 비교할 수 있습니다.

LLM 분석 공정 조건 정규화

배치 규모, 유틸리티, 주요 장비, 클린룸 요구사항을 CCD 관점으로 요약합니다.

산출물 CCD Package Draft

설계 요약, 주요 assumptions, 필요 deliverables, 채워진 입력 시트까지 한번에 정리합니다.

SERVICE VALUE

초기 협의 단계에서도 바로 검증 가능한 입력-출력 경험

유연한 입력 구조

엑셀 파일이 정해지지 않았더라도, 업로드와 폼 입력을 병행해 어떤 인터페이스가 더 적합한지 검토할 수 있습니다.

설계팀 관점의 요약

복잡한 제조 조건을 CCD 패키지 초안 구조에 맞게 정리해, 다음 단계 검토와 협업 속도를 높입니다.

향후 확장 용이성

실제 LLM API, 엑셀 파서, Google Drive 연동이 붙을 수 있도록 단계별 구조를 가볍게 분리했습니다.

WORKFLOW

프로토타입 사용자 흐름

01

입력 경로 선택

내 컴퓨터 파일 업로드, Google Drive 가져오기, 직접 옵션 입력 중 필요한 방식을 선택합니다.

02

데이터 정리

시트형 미리보기와 토글형 폼을 통해 프로젝트 조건을 확인하고, 부족한 값을 보완합니다.

03

CCD 패키지 초안 생성

분석 요약, 핵심 설계 요구사항, 예상 산출 문서, 입력 시트 반환용 데이터를 한 화면에 제공합니다.
