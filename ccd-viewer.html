ISPE Baseline Guide Vol 6의 시설 구역(Area)과 주요 설비(Equipment) 구조를 따르는 CCD JSON을 Three.js 캔버스에 시각화합니다. 마우스로 회전·줌인/아웃이 가능하며, 향후 LLM이 생성한 CCD 응답을 그대로 입력해 렌더링할 수 있도록 설계되었습니다.

Three.js Live Preview 설계 파라미터를 입력하고 생성 버튼을 눌러 주세요. 대기 중 · 모델 미생성

3D 객체 범례

  • CleanRoom_Wall (반투명 박스, 색조 = 압력)
  • Bioreactor (실제 형상 · dome + 다리 + 임펠러)
  • Chromatography / UFDF Skid
  • Filling Machine (실제 형상 · 아이솔레이터 + 노즐)
  • Buffer Tank (실제 형상 · ASME dished head)
  • HEPA 파티클 / 압력 캐스케이드 화살표

STEP 3

설계 평가 (Compliance Score)

Rule-based · concept-level

현재 배치를 4개 룰(압력 캐스케이드 / Grade 인접성 / 설비 밀도 / HEPA ACH)로 채점합니다. 실제 GMP 검증이 아니라 CCD 단계의 sanity check 수준이며, 점수는 의사결정 보조 용도입니다.

Overall Grade — 아직 평가되지 않았습니다
Weighted Score — 0 ~ 100
Scenario — 좌측 시나리오 셀렉트 기준
  • 「설계 생성」 버튼을 누르면 룰별 점수와 위반 사항이 여기 표시됩니다.
